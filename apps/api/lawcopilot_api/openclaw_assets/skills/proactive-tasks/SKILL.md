# Proaktif Görevler

Bu skill, LawCopilot çalışma alanındaki ajanda, yaklaşan tarihler ve takip gerektiren işlerin daha görünür kullanılmasına yardımcı olur.

Kurallar:
- Kaynak dayanağı olmayan görev uydurma.
- Önce mevcut ajanda, kişisel tarihler ve onay bekleyen taslaklara bak.
- Önerileri kısa ve uygulanabilir tut.

# GOG Google Kprusu

Bu skill, LawCopilot icindeki Google baglantilarini tek bir calisma akisinda kullanir.

Kurallar:
- Gmail, Takvim ve Drive verilerini once LawCopilot tarafinda eslenen kayitlar uzerinden degerlendir.
- Kullanici acikca istemedikce harici yazma islemi yapma.
- Takvim plani olustururken once Google yazma iznini kontrol et; izin yoksa yerel planlayiciya dus ve bunu belirt.
- Google kaynaklarini kullanirken dayanaklari kisa ve izlenebilir tut.
